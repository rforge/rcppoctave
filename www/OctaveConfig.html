Cached RcppOctave Configuration Paths

Usage

OctaveConfig(name, ..., reset = FALSE)

Arguments

name
Name of an RcppOctave path variable
...
extra names to be concatenated to the result with file.path. Only used when name is not missing.

Value

a list (if name is missing) or a single character string.

Description

Cached RcppOctave Configuration Paths

Examples

OctaveConfig()



$lib
                                LIBDIR                              OCTLIBDIR 
             "/usr/lib/i386-linux-gnu" "/usr/lib/i386-linux-gnu/octave/3.6.4" 

$include
                        INCLUDEDIR                      OCTINCLUDEDIR 
                    "/usr/include" "/usr/include/octave-3.6.4/octave" 

$modules
[1] "/tmp/RtmpWU2WQd/file466c34fe7f66/RcppOctave/modules"


OctaveConfig('lib')
